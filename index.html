The coinZdense project

The coinZdense project aims to provide a collection of post-quantum hash-based-signature libraries for different programming languages, aimed at utility blockchains, DEX blockchains and other blockchain technology thar requires or benefits from a key-reuse-by-design setup, both now and after quantum computing makes such designs problematic for ECDSA-using blockchains.

The coinZdense does not aim to be a usefull alternative for non-blockchain usage or for usage in blockchains that do not benefit from extensive key-reuse in any reasonable way.

Currently coinZdense is still in early stages of developement and it's a slow moving one-man project. It is however approaching a first MVP version for the first of the ten languages it aims to target: Python.
